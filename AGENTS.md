# Agent Environment: Swift Development
- **Compiler**: Swift 6.3 (Linux/Ubuntu)
- **Formatting**: Use `swift-format` if available; otherwise, follow Swift API Design Guidelines.
- **Testing**: Use the `swift test` command for package-based testing.
- **Project Structure**: This repository uses standalone scripts and `Package.swift` modules.
- **Focus Areas**: 
  - Strict Concurrency: Review for Sendable compliance.
  - SwiftData: Review schema definitions for Linux-compatible persistence.
  - SwiftUI: Note that Jules can review SwiftUI logic but cannot run native macOS previews.
